# RR-API-Wrapper
A Python Bot for Rival Regions

Current Features:
- simple gui
- auto perk upgrades

To use clone this repository, install Python (make sure pipenv is installed `pip install pipenv`) and then run from within the directory using shell.
```
pipenv install
pipenv run GUI.py
```
